# Datasets

This project uses the following publicly available datasets:

## 1. EyeQ Dataset
- Purpose: Retinal image quality assessment
- Labels: Good, Usable, Reject
- Usage: Training quality assessment models
- Source: https://github.com/HzFu/EyeQ

## 2. DRIMDB Dataset
- Purpose: Cross-dataset generalization testing
- Labels: Good, Bad, Outlier
- Source: https://www.kaggle.com/datasets/subhajournal/drimdb-diabetic-retinopathy-images-database
